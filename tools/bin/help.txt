typed-env <command>

Usage:

typed-env use [local | dev]                                  Use a type environment
typed-env get-env current-env-json change-env-json           Output changed environment variables
typed-env get-env-name current-env-json change-env-json      Output changed environment variable name
typed-env gen-env sourceFilePath tsconfigPath output         Check the file according to tsconfig and generate environment variables
